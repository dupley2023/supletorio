Evaluación Práctica de Recuperación - Desarrollo de una Aplicación Web de Registro de Usuarios


Tema: Desarrollo de una Aplicación Web de Registro de Usuarios



    Descripción de la Prueba:


  • Esta prueba, los estudiantes deberán desarrollar una aplicación web básica de registro de usuarios utilizando HTML, CSS y JavaScript.
  • Deberán aplicar los conceptos y técnicas que se han enseñado en la asignatura.
  • La aplicación debe permitir a los usuarios ingresar su nombre, dirección de correo electrónico y contraseña para registrarse. Los datos ingresados por los usuarios deben mostrarse en la página después del registro.

    Requisitos Específicos:

  • Crear un formulario de registro que incluya campos para el nombre, la dirección de correo electrónico y la contraseña del usuario.
  • Validar los campos del formulario para garantizar que se ingresen datos válidos antes de enviar el formulario.
  • Al hacer clic en el botón de registro, los datos del usuario deben mostrarse en la página de alguna manera (por ejemplo, en una lista o tabla).
  • Aplicar estilos CSS para mejorar la apariencia de la página.
  • Utilizar JavaScript para gestionar la lógica del formulario y la presentación de los datos.
  • Subir el proyecto a una plataforma de alojamiento web gratuita (por ejemplo, GitHub Pages o Netlify) para que el profesor pueda acceder y evaluar la aplicación.

    • Criterios de Evaluación:

  • Los estudiantes serán evaluados en función de los siguientes criterios:
  • Funcionalidad general de la aplicación web.
  • Validación de datos en el formulario.
  • Presentación de los datos registrados.
  • Calidad del código HTML, CSS y JavaScript.
  • Estilo y diseño de la página.
